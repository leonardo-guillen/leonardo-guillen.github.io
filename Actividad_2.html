1.HTML

Introduction

What is HTML?

HTML es un lenguaje de hipertexto, maneja un etiquetado que nos permite enmaquetar o encapsular el contenido de nuestra página web en elementos a los cuales podremos agregarles una configuración visual con CSS.

Explain the head and the body tags.

La etiqueta head tiene la particularidad de contener las cosas que no se mostraran visualmente en la página, tales como los metadatos, estilos o enlaces a archivos externos como el CSS. Por otro lado, el body contiene todo lo que se mostrara en nuestra página web de forma visual.

Basic Tags

Explain the section, article, aside, header, footer tag?

- section: La etiqueta section es un elemento más general, el cual es de ayuda para cuando no tenemos un elemento específico el cual pueda ayudarnos a representar algo, tal como un bloque general para el cual podríamos utilizar div, o para una muestra de texto donde en tal caso podría ser utilizado span. Por lo tanto, section lo tenemos para representar elementos más pequeños y específicos a los que podría representar un div.
- article: La etiqueta article es de ayuda para mostrar información auto-contenida, que es la forma de representar contenido repetitivo, tal como las tarjetas de representación de productos en una imagen o las tarjetas de representación de un mensaje, donde se representa un contenido distinto, pero se utiliza para representarlo con la misma UI por así decirlo.
- aside: La etiqueta aside tenemos dos formas principales de utilizarla, primeramente dentro de un article, la cual tendrá que encapsular elementos específicos los cuales estén relacionados con el mismo article. Por otro lado, es posible colocarlo dentro del propio body donde con ella representaremos contenido o elementos secundarios.
- header: La etiqueta header se utiliza para colocarla al inicio de las etiquetas que se utilizan para representar contenido importante, ya que se colocan en la parte superior para colocar elementos tales como un menú o similar.
- footer: La etiqueta footer es prácticamente lo mismo que la etiqueta header, pero de forma contraria, es más utilizada para ser colocada al final de nuestra página web y representar la información de contacto, derechos reservados o similares.

Lo mencionado anteriormente sobre cada etiqueta es el uso recomendable y sobre todo considerado buenas prácticas, evidentemente se podrían utilizar para otras cosas, pero el implementarlo como es mencionado es parte de tener buenas prácticas y un código más ordenado y entendible.

2.CSS

Introduction

What is CSS?

CSS es conocido por darnos la capacidad de agregar estilos a nuestras páginas y es considerado como una hoja de estilos de cascada, lo cual hace referencia a que los estilos que coloquemos en una etiqueta de html, algunos de estos son traspasados a las etiquetas dentro de esta, por ende se considera como una formación de cascada.

What is the purpose of CSS?

El propósito de CSS es permitirnos crear reglas para indicar como queremos mostrar el contenido de nuestra página web, de tal manera que mediante formatos específicos podremos representar toda nuestra información de una mejor forma visualmente.

How do you apply a CSS to an HTML (explain the 3 options)

Las tres formas que yo conozco para aplicar el CSS en nuestra página son:

  • Enlazar un archivo con extensión CSS, el cual suele contener la referencia hacia los ID, etiquetas o construcción de clases.
  • Agregar el estilo directamente en el head con la etiqueta style.
  • Agregar los estilos directamente a cada etiqueta con el atributo style.

What is a selector? What is a CSS rule?

Selector CSS rule
El selector en CSS es el que nos da la posibilidad de asignar reglas a todos los elementos del mismo tipo dentro de la página, si lo hacemos referenciando a p (p{}), lo que estariamos haciendo es asignar propiedades de CSS a todas las etiquetas de párrafo en nuestra página. Una regla en CSS es la definición de como se mostraran los elementos de nuestra página web con las propiedades de estilo, estas se consiguen de tal manera que se hace referencia a alguna etiqueta con el selector, id o se crea una clase la cual contendrá las propiedades de estilo.

Basic CSS

Explain a:

  • Tag Selector

    El selector en CSS nos permite hacer referencia a etiquetas html de forma general, si lo hacemos para la etiqueta p, se aplicará el estilo a todas las etiquetas p dentro de la página.

  • class Selector

    El selector de clase hará referencia a todos los elementos que tengan la misma clase dentro del atributo class en html, en CSS a estos se les hace referencia con un punto antes del nombre de la clase para poder aplicar los estilos .class.

  • Tag and class Selector

    El hacer referencia a la etiqueta y a la clase a la vez nos permite ser más específicos con la selección de un elemento, podremos tener 50 etiquetas div, pero si solo una tiene la clase a la que se hace referencia, solo serán aplicados los estilos a ese div.

  • Id Selector

    El selector de Id nos permite ser mucho más específicos, ya que el ID es único en cada elemento si este es colocado. No puede ser repetido y por ende al aplicar el selector por ID solo hacemos referencia a un elemento para aplicar los estilos.

What is a class? What is an id? How are they related to CSS?

Una clase nos permite crear una serie de estilos de forma general para reutilizar en distintos elementos de la página mediante el atributo class en html, mientras que en CSS se crean con un punto al inicio.

El ID es un atributo en html el cual se le asigna a una etiqueta y su contenido tendrá que ser único para cada elemento, de esta manera se podrá hacer referencia a un elemento específico desde CSS para aplicarle estilos.

En CSS podríamos relacionarlos de dos formas, primeramente es que ambas nos sirven para ser más específicos al momento de darle estilos a elementos de nuestra página, pero además se pueden mezclar para seleccionar elementos que tengan su ID dado y que además tengan la clase dada.
